# План тестирования проекта React Blog

## Цель тестирования
Проверить корректность ключевых пользовательских сценариев:
- Просмотр списка постов
- Просмотр выбранного поста
- Создание нового поста
- Валидация формы добавления поста
- Навигация между страницами

## Типы тестирования
| Тип | Что проверяется |
|-----|-----------------|
| Функциональное | Работоспособность функций блога |
| UX / Юзабилити тестирование | Удобство интерфейса и понятность интерфейса |
| Регрессионное | Отсутствие ошибок после исправлений |

## Среда тестирования
| Элемент | Значение |
|--------|---------|
| ОС | Windows 10 / Android |
| Браузер | Chrome (актуальная версия), Edge |
| Устройство | ПК / смартфон |
| Инструменты | DevTools, Console, Google Forms для обратной связи |

## Тестируемые функции
| ID | Функция | Описание |
|----|---------|----------|
| F1 | Просмотр списка постов | Отображение всех постов на главной странице |
| F2 | Открытие поста | Переход к содержимому выбранного поста |
| F3 | Создание поста | Возможность добавить пост через форму |
| F4 | Валидация формы | Ошибки при пустом заголовке / тексте |
| F5 | Навигация | Корректные переходы между страницами |
