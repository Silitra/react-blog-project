# React Blog

## 1. Описание проекта
React Blog — учебное клиентское веб-приложение, разработанное в рамках практической работы для освоения фреймворка React.  
Система предназначена для ведения блога: отображения опубликованных постов, добавления новых материалов и навигации по страницам.  
Приложение демонстрирует базовые принципы разработки SPA (Single Page Application), работу с состоянием и компонентным подходом.

---

## 2. Функциональные возможности
- просмотр списка постов;
- отображение содержимого выбранного поста;
- добавление новых материалов;
- навигация между страницами блога;
- использование компонентной структуры React.

## 3. Установка и запуск проекта

### 3.1. Требования
| Компонент | Версия / Требование |
|----------|---------------------|
| Node.js  | ≥ 18.x |
| npm      | ≥ 9.x |
| ОС       | Windows / Linux / macOS |


### 3.2. Инструкция установки
Клонировать репозиторий:
```bash
git clone https://github.com/Silitra/react-blog-project.git
cd react-blog-project
cd my-products-app
npm install
npm start
```
## 4. Примеры использования

### 4.1. Типовой сценарий пользователя

1. Пользователь открывает главную страницу приложения.
2. Видит список доступных постов блога.
3. Нажимает на выбранный пост и переходит на страницу с полным содержимым.
4. Переходит в раздел добавления поста.
5. Вводит заголовок и текст записи и сохраняет её.
6. Возвращается на главную страницу и видит новую запись в списке.

### 4.2. Команды npm

- `npm start` — запуск приложения в режиме разработки.
- `npm run build` — сборка оптимизированной версии приложения для продакшена.
- *(если используешь тесты)* `npm test` — запуск тестов.

## 5. Структура репозитория

```text
react-blog-project/
├── my-products-app/          # Исходный код React-приложения
```
